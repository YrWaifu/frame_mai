# Лабораторные работы по курсу "ИИ и фреймворки"

Репозиторий содержит 5 лабораторных работ, посвящённых базовым алгоритмам машинного обучения и их исследованию.  
Все работы выполнены в формате Jupyter Notebook, каждое решение включает:
- загрузку и анализ данных,
- построение baseline моделей,
- улучшение моделей подбором гиперпараметров,
- собственные реализации алгоритмов (где требуется),
- сравнение результатов.

## Используемые датасеты
- **Mushroom Classification** (Kaggle) — задача бинарной классификации.
- **Car Price Prediction** (Kaggle) — задача регрессии.

## Состав репозитория
- `lab1_knn.ipynb` — KNN (классификация и регрессия), улучшенный бейзлайн и собственная реализация.
- `lab2_regression.ipynb` — логистическая и линейная регрессия, улучшенные модели и собственные реализации.
- `lab3_decision_tree.ipynb` — решающее дерево, улучшенный бейзлайн и собственная реализация.
- `lab4_random_forest.ipynb` — случайный лес, baseline и улучшенные модели.
- `lab5_gradient_boosting.ipynb` — градиентный бустинг, улучшенный бейзлайн, итоговое сравнение алгоритмов.
