# Лабораторные работы по курсу "ИИ и фреймворки"

Репозиторий содержит результаты выполнения 5 лабораторных работ.
Цель: исследование и реализация базовых алгоритмов машинного обучения.

## Используемые датасеты
- **Mushroom Classification** (Kaggle) — задача бинарной классификации. https://www.kaggle.com/datasets/uciml/mushroom-classification
- **Car Price Prediction** (Kaggle) — задача регрессии. https://www.kaggle.com/datasets/amjadzhour/car-price-prediction

## Структура
*   `lab1_knn.ipynb` — Метод ближайших соседей (KNN).
*   `lab2_regression.ipynb` — Линейная и логистическая регрессия.
*   `lab3_tree.ipynb` — Решающие деревья.
*   `lab4_random_forest.ipynb` — Случайный лес (Random Forest).
*   `lab5_boosting.ipynb` — Градиентный бустинг (Gradient Boosting) и итоговое сравнение.

## Запуск
```bash
pip install -r requirements.txt
jupyter notebook
```
